# KeenKeeper — Keep Your Friendships Alive

## 📌 Project Description
This is a Friendship Analytics web application where users can track their interactions with friends such as calls, texts, and video chats. It also provides a visual representation of communication patterns using charts.

## 🚀 Technologies Used
- React.js
- Tailwind CSS
- Context API
- Recharts (for charts)
- JavaScript (ES6)

## ✨ Key Features
1. Track different types of interactions (Call, Text, Video)
2. Visual analytics using Pie Chart
3. Timeline view of all activities with date & type
